Projetos Realizados

Plataforma de Cursos Online (Front-end)
Desenvolvi uma plataforma de cursos online utilizando React.js e Tailwind CSS, proporcionando uma interface moderna e responsiva. O projeto inclui um sistema de navegação intuitivo, integração com API para listagem de cursos e um modo escuro para melhor experiência do usuário. Além disso, implementei animações suaves com Framer Motion para tornar a interação mais fluida e envolvente

API para Gestão de Tarefas (Back-end)
Criei uma API RESTful para um sistema de gestão de tarefas, utilizando Node.js com Express e banco de dados MongoDB. O back-end foi desenvolvido com autenticação JWT, validação de dados com Joi e testes automatizados para garantir a confiabilidade do sistema. A API permite que usuários criem, editem e excluam tarefas, além de acompanhar o progresso com status personalizados. Tudo foi estruturado seguindo boas práticas de arquitetura como MVC e princípios SOLID.

Redesign de um Aplicativo de Delivery (UX/UI)
Liderei o redesign de um aplicativo de delivery com foco na experiência do usuário. Através de pesquisas com usuários e análise de usabilidade, identifiquei pontos de melhoria e desenvolvi um novo design no Figma. A interface reformulada melhorou a navegação,reduziu o número de cliques necessários para realizar um pedido e otimizou o fluxo de checkout. O projeto resultou em um aumento na taxa de conversão e maior satisfação dos usuários.
